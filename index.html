About Me

Nama saya St Nur Syaridah, biasa dipanggil "Ridah" atau teman karib biasa memanggil saya "Edo" atau "Rido". Saya lahir pada 04 juni 2002 di Mamuju. Orang tua saya bernama Saharuddin dan Mardawiah. Saya merupakan anak kedua dari dua bersaudara. kakak saya Bernama Husain Hisbullah. Riwayat pendidikan saya sedari SD saya bersekolah di Sd Inpres Salubarana, kemudian SMP saya bersekolah di SMPS Salukayu IV, kemudian SMK saya bersekolah di SMKN 1 Papalang, dan seakarang Saya Masih melanjutkan pendidikan yakni di bangku perkuliahan dimana saya kuliah di Universitas Sulawesi Barat, dengan jurusan Teknik Informatika.
